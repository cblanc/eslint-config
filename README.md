# ESLint Typescript Config

## Usage

```bash
npm install --save-dev @cablanchard/eslint-config
```

In `eslintrc.js`

```json
{
  "extends": "@cablanchard"
}
```

In `package.json`

```json
{
  "scripts": {
    "lint": "eslint"
  }
}
```
